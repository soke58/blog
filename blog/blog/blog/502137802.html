VB filelist依次读取列表文件

VB6新建一个标准工程，添加一个filelist和timer控件。并在timer控件（其interval属性值设置为1000）添加如下代码：

Private Sub Timer1_Timer()

Dim i, f As Integer

  If File1.ListIndex = File1.ListCount - 1 Then '如果执行完退出

  Timer1.Enabled = False

    Else

    For i = 0 To File1.ListCount - 1

    If File1.Selected(i) <> 0 Then '判断如果FILELIST选择值为TURE时，

    i = i + 1

    File1.Selected(i) = i

    End If

    Next

End If

End Sub
